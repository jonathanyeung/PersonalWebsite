Guru Alpha

Guru is the answer to all of your questions! This Windows Phone app is a portal to Wolfram Alpha, an extremely powerful and extensive knowledge database. Use it to help with your physics homework, plot complex graphs, and discover facts about history and pop culture. Unlike other Wolfram Alpha client apps, this app uses an entirely native user interface for a quick and responsive experience. This app also tightly integrates with Cortana, allowing you to ask questions with your voice.

Technologies: XAML, C#, REST

Ozwego

Ozwego is a real-time multiplayer word game for the Windows Store that mimics the gameplay of the popular fast paced game of Bananagrams. Users can find friends to play against, or face other online users through a matchmaking system. The UI is optimized for both touch and mouse, so that both desktop and tablet users can play competitively. The backend is implemented on Windows Azure virtual machines, for a highly scalable solution that can support thousands of players.

Technologies: Windows Azure, TCP, XAML, C#

Point Reader

Point Reader is a Windows Phone app that lets you speed read through a method called rapid serial visual presentation. When reading text, people have to carefully scrutinize each word and reset their gaze when the end of a line is reached. Considering how many lines are in a novel, this slow down can be a significant amount of time! Point Reader flashes words one by one in a fixed position, so that your eyes remain focused in one spot. This results in faster reading speeds. With this technique, people read 30% faster on average.

Technologies: Silverlight, XAML, C#, OAuth
