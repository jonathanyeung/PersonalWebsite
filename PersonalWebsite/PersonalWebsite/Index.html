PROJECTS

Guru Alpha

Guru is the answer to all of your questions! This Windows Phone app is a portal to Wolfram Alpha, an extremely powerful and extensive knowledge database. Use it to help with your physics homework, plot complex graphs, and discover facts about history and pop culture. You can even tune your guitar with this app! The possibilities are endless. Unlike other Wolfram Alpha client apps, this app uses an entirely native user interface for a quick and responsive experience. This app also tightly integrates with Cortana, allowing you to ask questions with your voice.

Technologies: XAML, C#, REST

Ozwego

Ozwego is a real-time multiplayer word game for the Windows Store. Ozwego mimics the gameplay of the popular fast paced game of Bananagrams. Users can find friends to play against, or face other online users through the matchmaking system. Ozwego uses the Xaml/C# framework for its client implementation. The UI is optimized for both touch and mouse, so that both desktop and tablet users can play competitively against each other. The backend is implemented in C# on a Windows Azure Virtual Machine, for a highly scalable solution that can support thousands of players. The backend was designed to be multi-instance with a frontend load balancer to guarantee the service will have 99.9% uptime as per Azure's Service Level Agreement.

Technologies: Windows Azure, TCP, XAML, C#

Point Reader

Point Reader is a Windows Phone app that lets you speed read through a method called rapid serial visual presentation. When people read books, they have to carefully scrutinize each word and reset their gaze when the end of a line is reached. Considering how many lines are in a novel, this slow down can be a significant amount of time! Point reader flashes words one by one in a fixed position, so that your eyes can remain focused in one spot. This results in faster reading speeds. Researchers have found that with this technique, people are able to read 33% faster on average. With practice, people can read even faster.

Technologies: Silverlight, XAML, C#, OAuth
